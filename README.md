Top 10 Bosses Mais Difíceis de Elden Ring
Sobre o Projeto

Este projeto consiste no desenvolvimento de uma página HTML com o tema Elden Ring, apresentando um ranking dos bosses mais difíceis do jogo.

O projeto foi desenvolvido utilizando apenas HTML, sem uso de CSS ou JavaScript, conforme solicitado.
